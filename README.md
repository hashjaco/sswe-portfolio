# Next.js Portfolio
This is a simple portfolio website built with Next.js, showcasing my projects and skills. The website is designed to be fast, responsive, and easy to navigate.

## Features
- Responsive design
- Fast loading times
- SEO optimized
- Easy to navigate
- Dark mode support
- Customizable theme
- Lightweight and minimalistic design
- Supports multiple languages

## Technologies Used
- Next.js + React
- App Router
- Chakra UI
- Tailwind CSS
- TypeScript
- Framer Motion
- React Icons
- Supabase
- React Query
- React Hook Form
- Zod
- react-three-fiber
